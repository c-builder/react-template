## 一套功能完整的React框架，能够快速构建React项目
* React脚手架工具
* React前后端同构渲染
* Koa
* ES6、ES7语法支持
* 单向数据流：基于Alt实现
* i18n国际化
* LESS、SASS、PostCss
* webpack
* 雪碧图
* Browsersync能让浏览器实时、快速响应您的文件更改,同时支持React热插拔

### 安装依赖包
```javascript
npm install
```

### 启动
```javascript
npm start
```
#### 注意
> 本地运行时，先把本机hosts文件配置为：
```
127.0.0.1          dev.cbuilder.com
```
